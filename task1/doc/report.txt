1.
===
El código resuelve un sistema de ecuaciones lineales usando la rutina de la
librería LAPACK llamada `dgesv`. De acuerdo a la documentación de la librería,
resuelve el sistema de ecuaciones haciendo una descomposición LUP, que
garantiza que el algoritmo funcione para cualquier matriz de entrada.

2.
===
Sí, existen distintos métodos para resolver un sistema de ecuaciones lineales.
Por ejemplo, existe una fórmula que se puede usar para resolverlo sin tener
que operar con las matrices. Sin embargo, depende de que se cumplan ciertas
condiciones. Otro método es resolver el sistema usando el algoritmo de
reducción de filas (Gaussian elimination), el cual construye las matrices L y U
para, finalmente, calcular la solución al sistema. También existe un algoritmo
que implementa el método anterior mediante recursión. Por último, también es
posible utilizar procesos estocásticos para obtener una solución relativamente
buena sin tener que calcular la descomposición (Cholesky decomposition).

3.
===
- Fórmula: require la computación de muchos determinantes, la cual es
computacionalmente costosa. En práctica no es apropiado.
- Eliminación Gaussiana: requiere de O(2/3n^3) operaciones en punto flotante
para calcular la descomposición y O(n^3) operaciones para multiplicar las
matrices resultantes y obtener la solución al sistema.
- Recursión: consiste en realizar LUP recursivas y multiplicaciones. En
esencia, es un algoritmo O(n^3), como la eliminación gaussiana.
- Cholesky decomposition: llega a ser casi dos veces más eficiente que una
descomposición LU normal (cualquiera de las dos anteriores). Aunque no siempre
es posible computarlo.

4.
===
Probablemente el mejor algoritmo para paralelizarse a gran escala sea el que
se basa en procesos estocásticos. En esencia, es una simulación de Monte Carlo,
las cuales son altamente paralelas. Por la contra, los métodos de reducción y
recursión son difíciles de paralelizar. No es sencillo realizar una
descomposición de dominio sencilla para poder paralelizar los algoritmos a gran
escala.
